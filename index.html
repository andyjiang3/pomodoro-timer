Pig-Latin Web Page Translator

Translate HTML files to working Pig Latin HTML file using standard translation rules.

How It Works

Scans through file and ignore HTML tags, only translating the content to Pig Latin.

Skip pass items that fits the pattern of &text;. These are HTML character entitites' name and must be kept the same.

Determine content of the HTML file that is visible to users and translate it based on Pig Latin rules

Pig Latin Rules

1.

A word to be translated is a consecutive sequence of letters (a-z, A-Z) or apostrophes, containing at least one vowel or y in a vowel position.

2.

If a word starts with a vowel, the Pig Latin version is the original word with "way" added to the end

3.

If a word starts with a consonant, the Pig Latin version transfers all consonants up to the first vowel to the end of the word, adding "ay" to the end.

4.

The letter 'y' should be treated as a consonant if it is the first letter of a word, but treated as a vowel otherwise.

5.

If the original word is capitalized, the new Pig Latin version of the word should be capitalized in the first letter.
